# 📊 Painel de Saúde Pública

Projeto desenvolvido para a disciplina de Ciência dos Dados.

O sistema utiliza:

- Python
- Streamlit
- PostgreSQL
- Docker
- SQLAlchemy
- Pandas

O objetivo do projeto é analisar dados de atendimentos de saúde pública através de um dashboard interativo.

---

# 🚀 Funcionalidades

- Visualização de atendimentos
- Gráficos por diagnóstico
- Gráficos por bairro
- Filtro interativo
- Casos por dia
- Análise por faixa etária
- Banco de dados PostgreSQL integrado
- Aplicação containerizada com Docker

---

# 🛠 Tecnologias utilizadas

- Python 3.10
- Streamlit
- PostgreSQL
- Docker
- Pandas
- SQLAlchemy

---

# ▶️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/marianadj901/painel-saude-publica.git
```

Entre na pasta do projeto:

```bash
cd painel-saude-publica
```

Execute o Docker:

```bash
docker compose up --build
```

Depois acesse no navegador:

```txt
http://localhost:8501
```
